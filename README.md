# Customer Shopping Data Analysis

## Project Overview

This project performs a multi-dimensional exploratory data analysis on customer shopping data to understand purchasing behavior, revenue distribution, and sales performance across shopping malls, product categories, payment methods, and customer demographics.

---

## Dataset

* **Dataset:** Customer Shopping Data
* **File:** https://www.kaggle.com/datasets/mehmettahiraslan/customer-shopping-dataset
---

## Objectives

* Check data quality
* Convert invoice dates to datetime format
* Extract Year and Month
* Analyze spending across multiple business dimensions
* Build pivot tables for revenue analysis
* Analyze yearly revenue trends
* Compare male and female spending behavior
* Generate business insights through visualizations

---

## Data Preparation

The following preprocessing steps were completed:

* Loaded the dataset
* Checked missing values and data types
* Converted `invoice_date` to datetime format
* Extracted Year and Month columns

---

## Analysis

The project includes:

* Spending by Category
* Spending by Shopping Mall
* Spending by Payment Method
* Spending by Gender
* Average Purchase Amount by Category
* Revenue Ranking
* Shopping Mall vs Category Pivot Table
* Yearly Revenue Trend
* Male vs Female Spending Comparison

---

## Visualizations

* Pivot Table Heatmap (Shopping Mall vs Category)
* Yearly Revenue Trend Line Chart
* Grouped Bar Chart (Gender vs Category Spending)
